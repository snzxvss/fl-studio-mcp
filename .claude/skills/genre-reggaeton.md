# Reggaeton Production Reference

## Tempo

**85-100 BPM** (standard: 95 BPM). Modern tracks up to 110 BPM.

## The Dembow Pattern (1 bar = 4 beats)

The signature reggaeton rhythm. All times in quarter notes (beat 1 = 0.0).

### Kick (MIDI 36)

| Time | Velocity | Duration |
|------|----------|----------|
| 0.0  | 0.9      | 0.25     |
| 1.0  | 0.85     | 0.25     |
| 2.0  | 0.9      | 0.25     |
| 3.0  | 0.85     | 0.25     |

## Bass Pattern

Follows chord roots. 808 sub-bass feel, MIDI 33-45 range.

**For Am chord (root A2 = MIDI 45):**

| Time | MIDI | Duration | Velocity |
|------|------|----------|----------|
| 0.0  | 45   | 1.5      | 0.9      |
| 2.0  | 45   | 1.0      | 0.85     |
| 3.0  | 45   | 0.5      | 0.7      |

**Minimal 808 variant:** Only beats 1 and 3 (times 0.0, 2.0), duration 1.75, velocity 0.95.

Change root note to match each chord:
- Am → A2 (45), Dm → D3 (50), G → G2 (43), C → C3 (48), F → F2 (41), E → E2 (40)

## Song Structure

| Section      | Bars | Elements |
|-------------|------|----------|
| Intro        | 4-8  | Filtered beat, hook motif |
| Verse 1      | 8    | Drums + bass + sparse chords |
| Pre-chorus   | 4-8  | Build tension, add hats/perc |
| Chorus       | 8    | Full groove, all elements |
| Post-chorus  | 4    | Perreo drop, heavy bass |
| Verse 2      | 8    | Variation on verse 1 |
| Bridge/Break | 4    | Strip drums, melodic focus |
| Final Chorus | 8    | Full energy, layered |
| Outro        | 4    | Fade or abrupt stop |

## Quick Start Recipe

To build a basic reggaeton beat from scratch:

1. Set BPM to 95
2. Send the dembow drum pattern (kick + snare + hi-hat) — 1 bar, loop
3. Add chord progression 1 (Am-Dm-G-C) — 4 bars
4. Add bass following chord roots — 4 bars
5. Add a simple melody using A minor pentatonic — 4 bars
6. Copy the 4-bar pattern to fill the song structure
